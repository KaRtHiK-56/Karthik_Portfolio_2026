## ⚙️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/KaRtHiK-56/Karthik_Portfolio_2026.git
cd Karthik_Portfolio_2026
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm start
```

Build for production:

```bash
npm run build
```
